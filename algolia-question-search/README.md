# Algolia Question Search Demo

This Vite app demonstrates searching the `question_objects` Algolia index with `react-instantsearch` v7 and allows restricting searchable attributes.

## Run locally

```bash
cd viteApps/algolia-question-search
npm install
npm run dev
```

## Build for S3/CloudFront

```bash
npm run build
```

The production build outputs to `dist/`.
